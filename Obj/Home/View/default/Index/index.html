公司简介

广州宇脉电子科技有限公司始于2003，是一家专业的单片机开发、电子产品定制开发及生产的企业，本公司单片机产品被认定为广东省高薪技术产品，并拥有多项发明专利、实用新型专利、外观专利、软件著作权。是广东省守合同重信用企业，有着16年深厚的技术沉淀。本公司自主品牌为：宇脉、宇脉单片机。自主研发的产品涉及智能自助产品系列控制系统、汽车维修设备系列控制系统、智能家居系列控制系统、数据采集系列控制系统…………
